Question-1: What are the different ways to select an element in the DOM?

1. document.getElementById(id): Selects an element by its id.

2. document.querySelector(selector): Selects the first element matching a CSS selector.

3. document.querySelectorAll(selector): Selects all elements matching a CSS selector.

4. document.getElementsByClassName(className): Selects elements by class name.

5. document.getElementsByTagName(tagName): Selects elements by tag name.

6. document.getElementsByName(name): Selects elements by name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerHTML: Gets or sets the HTML content inside an element, including tags.

2. innerText: Gets or sets the visible text content, excluding hidden elements and respecting CSS styles.

3. textContent: Gets or sets the text content, including hidden elements, but ignores HTML tags.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where you attach a single event listener to a parent element to handle events for all its child elements. This is useful for dynamically added elements or when you have many child elements.

Example:

document.getElementById("parent").addEventListener("click", function(event) {
                    if (event.target.tagName === "BUTTON") {
                        console.log("Button clicked:", event.target.textContent);
                    }
                });
Question-4: What is event bubbling in the DOM?

Event Bubbling is the process where an event propagates from the target element up through its ancestors in the DOM tree. For example, if a button inside a div is clicked, the click event will first trigger on the button, then on the div, and so on up to the document.

You can stop bubbling using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create: Use document.createElement(tagName) to create a new element.

Example: const div = document.createElement("div");

2. Add: Use parentElement.appendChild(newElement) to add an element as the last child of a parent.

Example: document.body.appendChild(div);

3. Remove: Use parentElement.removeChild(childElement) to remove an element.

Example: document.body.removeChild(div);
